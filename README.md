# Leave Management System

This is a web-based Leave Management System developed to manage student and staff leave requests efficiently. The system allows users to apply for leave and enables staff/admin to review and manage requests.

## Features

* Student Login System
* Staff Login System
* Apply Leave Form
* View Leave Status
* Staff Department Selection
* Leave Approval / Management
* Database Integration

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Project Structure

* index.php – Home page
* student_login.php – Student login page
* student_dashboard.php – Student dashboard
* staff_login.php – Staff login page
* staff.php – Staff dashboard
* staff_department.php – Department selection
* leave_form.php – Leave application form
* submit.php – Form submission logic
* view_leave.php – View leave status
* db.php – Database connection
* leavedb.sql – Database file
* style.css – Styling
* script.js – JavaScript functions

## How to Run the Project

1. Install XAMPP/WAMP
2. Place the project folder inside `htdocs`
3. Start Apache and MySQL
4. Open phpMyAdmin and import `leavedb.sql`
5. Open browser and type:
   http://localhost/leave_project
